Unit description and texts DP assessment(s) for unit

【内容】本单元的内容主要是针对 DP BSL 的学生设计的，单元内容主要围绕"传统节日与习俗"这一话题展开，学生不仅会学习相关语言知识，还会了解到：1、节日习俗在现代生活的冲击下产生了新形式，更加符合现代人的生活习惯；2、无论时代如何变化，中国传统节日维系亲情的内核不变。

5C：互相交流习俗文化（沟通）；理解中国文化中对亲情的重视（文化）；运用一张海报，介绍相关的习俗；对比不同关联词使用情景的异同；了解所在社区的节日习俗。

【培养學科能力】1、培养学生看图说话、组织语言的能力；2、培养学生运用博客文体有条理地阐述自己观点的能力；3、培养学生总结段落大意的能力。

【教材】1、重阳节习俗知多少（网页文章）2、春节习俗新元素大盘点 3、中秋节4、异乡过节也很幸福

评估（SA）

写作：学生将在学习的过程中不断收集关于传统习俗演变的观点并完成一篇博客写作，主要有两个任务组成：

1、通过学习《春节习俗新元素》和《异乡过节也很幸福》，收集关于传统习俗利弊的观点、词汇、句式，制作成思维导图。

2、作为网友，完成一篇探讨传统文化应当改变与否的博客写作

口语：能够通过一张视觉引导图片向全班作报告，运用所学的组织语言的方式，介绍该节日习俗以及背后的文化。
